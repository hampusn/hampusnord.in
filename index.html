Ett par exempel på vad jag har gjort.

  • jquery.sbname.js

    sbname på Github.

    Ett jQuery-plugin som använder Yahoo Query Language för att hämta namnet på en artikel från Systembolaget.

    YQL används på grund av restriktioner hos systembolaget.se gällande http-frågor via ajax. YQL har inga restriktioner och vi kan alltså gå via den tjänsten för att man inte ska behöva något annat än det här pluginet.

    Med det här projektet vill jag dels visa att jag behärskar jQuery men även att jag inte är rädd för att kommentera min kod.

  • hampusn-current-template

    Current Template på Github.

    WordPress-plugin för temautvecklare. Pluginet visar vilken temamall som används på sidan du befinner dig på för att du snabbt ska kunna se att rätt mall laddas in via WordPress mallsystem.

    Det här började som ett litet hjälpskript som användes under ett kundprojekt men skrevs sedan om på fritiden i och med att jag såg ett behov för pluginet.

    Här vill jag mest ge lite mer bredd och som ett komplement till mitt andra exempel.
